# Agent Instructions

Follow the project conventions in [CLAUDE.md](CLAUDE.md).

Important Tailwind CSS utilities must use the Tailwind v4 trailing `!` syntax:

- Correct: `size-9!`, `rounded-[10px]!`, `hover:bg-surface-container-high!`
- Incorrect: `!size-9`, `!rounded-[10px]`, `hover:!bg-surface-container-high`
